## 3. 消息链路说明

- 用户在公众号聊天窗口输入命令文本（如 `help`、`commit ...`）
- 微信服务器把消息以 XML 的方式 POST 到后端回调接口
- 后端解析 XML，得到 `FromUserName`（用户标识）与 `Content`（用户输入）
- 后端返回微信规定格式的 XML 回复文本
